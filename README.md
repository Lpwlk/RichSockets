
# RichSockets

  Python3 CLI app based on socket and rich modules for networking terminal interface rendering.

***

# Usage

 > Note : To install required packages from PyPi.org, *use pip (Windows machine) machine or pip3 (UNIX machines).*
 > Minimal Ptyhon3 version required for every dependencies : *3.12.1*

```
pip install rich
pip install numpy as np
pip install richsockets
```
